# Examples

Real output from a real run (June 9, 2026 — 30 videos across 12 AI-focused channels),
included so you can see what you're building before you set anything up.

- **[example-briefing.html](example-briefing.html)** — a finished daily brief.
  [▶ View it in your browser](https://htmlpreview.github.io/?https://github.com/agrotisnicolaos/agr-informed/blob/main/examples/example-briefing.html)
  (or download and double-click it). The full anatomy of every section is explained
  in the [Setup Guide](../docs/GUIDE.md#reading-your-brief).
- **[example-briefing.json](example-briefing.json)** — the structured analysis behind
  that report, exactly as Claude writes it each morning.
- **[wiki/](wiki/)** — two sample second-brain topic pages showing how findings
  accumulate: a dated timeline, a living "current state", and explicitly flagged
  contradictions.

Your own reports, briefings, and wiki stay private on your machine — these examples
are the only ones that ship with the project.
